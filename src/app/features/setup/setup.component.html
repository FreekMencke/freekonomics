Setup

Freekonomics uses the PSD2 API to connect with your bank accounts so we can create an up-to-date overview of your wealth.

There is only one caveat: not all banks, and not all account types support PSD2. We can work around this by letting the user create custom accounts themselves.

We use Nordigen to easily connect with the PSD2 API. In order to use Freekonomics, you will need an Nordigen account.
  • To get started with Freekonomics, you first have to create a Nordigen account.
  • After creating an account, you need to create a User Secret.
Please enter the User Secret below. This secret will only be saved on this device to keep it safe.

Freekonomics itself only stores data on this device, for other concerns about your data, please check your Nordigen account.
